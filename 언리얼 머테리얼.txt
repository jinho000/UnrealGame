- PBR 기반 렌더링
물리기반 렌더링
빛이 물체표면에 반사되는 경로를 물리적인 공식을 이용해 계산
그 결과를 머티리얼을 통해 표현함
-> 물리적인 계산을 통해 물체를 표현

- PBS
물리 기반 쉐이더
물리공식을 이용하여 수치값 변경을 통해 하나의 쉐이더만으로 다양한 재질표현 가능
물리기반이 아닐경우 각 재질마다 쉐이더를 따로 만들어야함

- 머테리얼
물체를 표현할 쉐이더와 물체에 대한 수치값을 저장한 리소스
하나의 머테리얼은 하나의 재질만 표현해야함
여러색을 사용할 경우 머테리얼은 각 색마다 따로 제작해야함
-> 빨강색 머테리얼, 파랑색 머테리얼, ...


- PBR기반 쉐이더의 프로퍼티들

1 Base Color(Albedo)
물체가 가진 고유색을 결정

Diffuse: 난반사
Specular: 정반사

2 Metallic
물체를 금속성으로 표현할지 비금속성으로 표현할지 설정
금속의경우 빛을 받았을때 스스로의 컬러를 반사시킴
비금속의 경우 빛을 받았을때 받의 빛의 컬러를 반사시킴

* 메시의 UV
텍스쳐를 매시에 입히기위해 텍스쳐의 좌표에 매핑되는 매시의 좌표값
0 ~ 1로 표현됨

3 Roughness
물체 표면의 거친느낌 표현
유리나 거울처럼 물체의 표면이 매끄러운경우 0
물체의 표면이 거친경우 1
표면이 매끄러운경우 반사효과도 생김

4 Normal Map
물체는 메시의 폴리곤을 많이 쓸수록 디테일한 표현이 가능
-> 폴리곤을 많이 쓸수록 성능 저하가 심하다
폴리곤을 적게 쓰면 디테일한 표현이 어려움
-> 물체 표면의 굴곡을 표현하는 노말맵을 따로 만들어 재질을 표현, 
성능을 올리며 디테일한 작업 가능

물시의 UV좌표에 대응하는 텍스쳐값이 필요 하므로 텍스쳐 형태로 제작됨


5 Emissive Color
자체 발광 효과
스스로 빛을 뿜어내는 재질을 표현
0 ~ 1까지의 값을 받거나 텍스쳐를 받기도함
HDR라이팅이 지원되어 1보다 큰 값을 입력받아 더 강한 발광효과를 표현할 수 있음
포스트프로세싱 효과를 이용 화사한 효과 가능
