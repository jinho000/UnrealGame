- UMG모듈
언리얼 C++에서 UI 위젯을 사용하기위한 모듈
UMG모듈은 따로 분리되어있어서 모듈을 추가해줘야함
build.cs -> PublicDependencyModuleNames에서 "UMG"모듈 추가


- UPROPERTY - meta
메타데이터 지정자 옵션
언리얼 에디터에 대한 제어를 할 때 사용
에디터에 변수를 노출시킬때 코드와 이름을 달리하거나
변수 입력값을 특정범위로 제한함

BindWidget
C++코드에서 만든 위젯관련 변수를 실제 위젯 블루프린트와 연동
이 클래스를 상속한 블루프린트에서 반드시 구현해야함
-> 블루프린트에서 해당 변수값이 반드시 존재 해야함(코드의 변수이름과 동일해야함)
코드에서 변경된 값이 위젯 블루프린트에 반영됨

- 위젯생성
GameModeBase에 생성할 위젯 클래스와 생성된 위젯을 저장할 변수 설정
언리얼 에디터에서 상속받은 블루프린트는 생성할 위젯 블루프린트를 설정해줘야함
게임이 시작되면 위젯을 생성(CreateWidget)하고 뷰포트(AddToViewport)에 추가


- 위젯에서 버튼 클릭시 함수 연결
위젯의 경우 begin play 함수가 존재하지 않음
대신 native construct함수를 이용해 위젯 생성시 버튼 클릭 이벤트 함수를 연결
